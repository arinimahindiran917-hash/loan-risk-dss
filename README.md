# Loan Risk Decision Support System

This project develops a machine learning-based decision support system for predicting loan default risk using the UCI Default of Credit Card Clients dataset.

## Project Features

- Data preprocessing and cleaning
- Multiple machine learning models
- Logistic Regression
- Random Forest
- Gradient Boosting
- LightGBM
- CatBoost
- Model comparison and evaluation
- ROC Curve and Confusion Matrix
- SHAP Explainability
- Feature Importance Analysis
- Streamlit Dashboard for real-time prediction
