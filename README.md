# Loan Risk Decision Support System
This project predicts the probability of credit card default using machine learning techniques.
# Dataset
The project uses the UCI Credit Card Default Dataset (Yeh and Lien, 2009), which contains financial information such as credit limit, repayment history, bill amounts, and payment behaviour.
# Features
- Data preprocessing and cleaning
- Multiple machine learning models:
  - Logistic Regression
  - Random Forest
  - Gradient Boosting
  - LightGBM
  - CatBoost
- Model evaluation using:
  - Accuracy
  - Precision
  - Recall
  - F1 Score
- SHAP explainability for feature importance
- Streamlit-based decision support system
# Application
A Streamlit application is developed where users can input financial data and receive a prediction of loan default risk.
# Project Structure
- src/ → Data preprocessing and model scripts
- models/ → Trained models
- app.py → Streamlit application
- evaluation_dashboard.py → Model evaluation dashboard
# Note
The dataset is not uploaded due to size limitations. It is publicly available from the UCI Machine Learning Repository.
